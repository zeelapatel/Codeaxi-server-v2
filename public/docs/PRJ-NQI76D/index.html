Get Project

URL: /projects/:id

Method: GET

Parameters:

  • id (string) - The ID of the project.

Response:

{
  "id": "string",
  "name": "string",
  "description": "string",
  "projectContext": "object",
  "generatedApiDocs": "array"
}

Example:

fetch('/projects/123', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
}).then(response => response.json())
  .then(data => console.log(data));
